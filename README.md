# 📡 Placement of Access Points in an Ultra-Dense 5G Network

## 📌 Overview

This project focuses on optimizing the placement of access points (APs) in ultra-dense 5G networks to achieve uniform coverage and reduce power wastage. Traditional layouts such as circular and hexagonal patterns often lead to overlapping coverage or dead zones.

To address this, the project proposes the use of **Centroidal Voronoi Tessellation (CVT)** combined with **adaptive power optimization** to improve network efficiency.

---

## 🚀 Key Features

* Uniform access point distribution using CVT
* Reduction of overlap and coverage gaps
* Adaptive power control for boundary nodes
* Improved spectral efficiency
* Approximately **15% reduction in power wastage**
* Stable and uniform network performance

---

## 🧠 Algorithms Used

### 1. Hexagonal Packing

Used as a baseline model for comparison with traditional network layouts.

### 2. Centroidal Voronoi Tessellation (CVT)

Optimizes AP placement by positioning them at the centroid of coverage regions and dynamically adjusting positions and power.

## 📊 Methodology

1. Initialize service region and transmission radius
2. Apply traditional placement (square & hexagonal)
3. Apply CVT-based optimization
4. Perform iterative centroid adjustment
5. Evaluate performance using:

   * Coverage analysis
   * Overlap detection
   * Spatial uniformity
   * Energy convergence

---

## 📈 Results

* Reduced power wastage by ~15%
* Improved uniformity of AP distribution
* Minimized signal overlap and interference
* Achieved near-complete coverage
